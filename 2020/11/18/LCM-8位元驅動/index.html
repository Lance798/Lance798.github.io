《FPGA研究筆記 三》 LCD模組 8位元驅動

發表於 2020-11-18 更新於 2020-12-14 分類於 FPGA研究筆記 Valine：

目標

使用FPGA，透過8bit傳輸介面來驅動LCD模組。

相關知識

LCD模組

KTM-626上安裝的LCD型號是WG14432B5，可以顯示中文字，為繪圖型LCD。
寬度144畫素、高度32畫素。

名詞說明

  • AC 位址計數
  • BF 忙碌旗標
  • DDRAM 顯示資料暫存器
  • CGRAM 自定義字型暫存器
  • HCGROM 半寬字型記憶體(ASCII)
  • CGROM 中文字型記憶體(BIG-5)

重要接腳說明

  1. RS 暫存器選擇 和 RW 讀/寫控制選擇
    RS RW 簡介
    0 0 寫入指令碼於指令暫存器。
    0 1 讀取BF(忙碌旗標)和AC(位址計數器)的內容。
    1 0 將資料寫入DDRAM(資料暫存器)。
    1 1 從DDRAM讀取資料。
  2. EN LCD致能接腳
    輸入一正脈波(0 -> 1 -> 0)即可完成寫入資料。
  3. D0 ~ D7 資料匯流排
    輸入指令碼

LCM 8位元控制器

  • 接腳圖

    • inLCM_CLK

      時脈輸入(50MHz)

    • inLCM_RESET

      重置輸入

    • inLCM_DB[7..0]

      字元編碼/指令碼輸入

    • Action[1..0]

      控制動作

      Action=0：執行初始化

      Action=1：將inLCM_DB的字元編碼輸入DDRAM

      Action=2：執行inLCM_DB的指令碼

    • outLCM_EN

      LCM致能輸出

    • outLCM_RW

      LCM讀/寫控制選擇RW輸出

    • outLCM_RS

      LCM暫存器選擇RS輸出

    • outLCM_DB[7..0]

      Data Bus輸出

    • outLCM_isDone

      輸出結束isDone=1，否則為0
